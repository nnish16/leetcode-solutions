# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in **leetcode-solutions**, please report it responsibly:

1. **Do not** open a public issue.
2. Email [nnish1611@gmail.com](mailto:nnish1611@gmail.com) with details.
3. Include steps to reproduce the issue if possible.
4. Allow reasonable time for a fix before public disclosure.

| Version | Supported |
|---------|-----------|
| latest  | Yes       |
